# Tkinter Alert Spammer

A simple alert spammer using Python's Tkinter library.

## Features
- Create multiple alert windows
- Customize messages
- Set number of alerts

## Requirements
- Python 3.x
- Tkinter

## Usage
1. Clone/download repo
2. Run: **python alerts.py**
